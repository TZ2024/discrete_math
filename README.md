# Discrete Math Interactive App (Chapter 6)

This repository contains a Streamlit app for Chapter 6 on **Relations**.

## Contents

Current sections in the app:
- **Basics**: ordered pairs and relation tables
- **Modeling**: properties, digraphs, adjacency matrices, and transitive closure
- **Operations**: composition of relations
- **Advanced Applications**: topological sorting and equivalence relations

## Run Locally

```bash
cd Desktop/GSRA/discrete_math
python3 -m pip install --user -r requirements.txt
python3 -m streamlit run Home.py
```
